## Get involved

### Developer mailing list

The openvpn-devel mailing list is the primary channel for:
- Submitting patches for review
- Technical discussions about OpenVPN development
- Architecture and design decisions
- Release planning and coordination

Subscribe at: [https://lists.sourceforge.net/lists/listinfo/openvpn-devel](https://lists.sourceforge.net/lists/listinfo/openvpn-devel)

### IRC chat

For quick feedback and real-time discussions, join the #openvpn-devel channel on irc.libera.chat.

<Info>
  You need to be logged in to Libera to join the channel. See the [Libera Chat registration guide](https://libera.chat/guides/registration) for instructions.
</Info>

## Reporting issues

<Note>
  Issues for OpenVPN Access Server, OpenVPN Connect, or other commercial products should be reported directly to OpenVPN Inc. at [https://support.openvpn.net](https://support.openvpn.net)
</Note>

For the open source OpenVPN project:

1. Check existing issues on [GitHub](https://github.com/OpenVPN/openvpn/issues)
2. Create a new issue with:
   - Clear description of the bug
   - Steps to reproduce
   - OS and OpenVPN version information
   - Expected vs actual behavior
